Dicas para estudar HTML


Para estudar HTML, é necessário bastante prática, entender as divisões básicas (Head, body), ter uma boa IDE que dê sugestões para facilitar o processo de escrita do código, entre outras coisas.

Um bom inglês facilita bastante o aprendizado, pois é mais fácil pensar em palavras para as coisas que se quer fazer no código. Por fim, pesquisas na internet e ver códigos prontos também pode ser bastante útil.
